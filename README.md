# Sales Dashboard in Excel

## Project Overview

This Sales Dashboard project was built using Microsoft Excel to analyze sales performance and provide actionable business insights. The dashboard presents key metrics, trends, and visualizations that help track revenue, orders, and customer performance efficiently.

## Objectives

* Monitor overall sales performance.
* Track revenue and profit trends.
* Identify top-performing products and regions.
* Analyze customer purchasing behavior.
* Support data-driven business decisions.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Excel Formulas (SUMIFS, COUNTIFS, XLOOKUP)

## Key Performance Indicators (KPIs)

* Total Sales
* Total Orders
* Total Profit
* Average Order Value
* Top Selling Products
* Regional Sales Performance

## Dashboard Features

* Interactive slicers for filtering data.
* Dynamic charts for sales analysis.
* KPI cards for quick insights.
* Monthly and regional sales trends.
* User-friendly and visually appealing design.

## Insights Generated

* Identified top-performing products and regions.
* Tracked monthly sales growth patterns.
* Analyzed customer contribution to revenue.
* Highlighted opportunities for improving sales performance.

## Project Outcome

The dashboard provides a comprehensive view of business sales performance, enabling stakeholders to make informed decisions and improve operational efficiency.
